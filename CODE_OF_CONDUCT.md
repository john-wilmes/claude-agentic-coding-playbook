# Contributor Covenant Code of Conduct

This project has adopted the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

All contributors, maintainers, and participants are expected to uphold these standards. By participating in this project, you agree to abide by its terms.

## Reporting

Report violations to the repository maintainers via a private GitHub message or the email listed in the repository contact information.
